VSCode Favorite Settings README

1. Open your terminal.
2. Enter the command: nano ~/.bashrc
3. Scroll down to the end of the file.
4. Copy and paste the contents of Favorite settings.txt
5. Press CTRL+O (WriteOut) to save the changes.
6. Press CTRL+X to exit nano.
7. Restart your terminal or source the .bashrc file with the command: source ~/.bashrc
8. Your terminal prompt should now include your favorite VSCode settings.
